<p>&lt;h1&gt;Setup and Installation&lt;a class=”headerlink” href=”#setup-and-installation” title=”Permalink to this headline”&gt;¶&lt;/a&gt;&lt;/h1&gt;
&lt;div class=”section” id=”step-1”&gt;
&lt;h2&gt;Step 1&lt;a class=”headerlink” href=”#step-1” title=”Permalink to this headline”&gt;¶&lt;/a&gt;&lt;/h2&gt;
&lt;p&gt;Clone the repository. Then run &lt;code class=”docutils literal notranslate”&gt;&lt;span class=”pre”&gt;pipenv&lt;/span&gt; &lt;span class=”pre”&gt;shell&lt;/span&gt; &lt;span class=”pre”&gt;.&lt;/span&gt;&lt;/code&gt;&lt;/p&gt;
&lt;/div&gt;
&lt;div class=”section” id=”step-2”&gt;
&lt;h2&gt;Step 2&lt;a class=”headerlink” href=”#step-2” title=”Permalink to this headline”&gt;¶&lt;/a&gt;&lt;/h2&gt;
&lt;p&gt;To make sure Earth Engine API is configured with your Google Drive, do the following:&lt;/p&gt;
&lt;div class=”highlight-bash notranslate”&gt;&lt;div class=”highlight”&gt;&lt;pre&gt;&lt;span&gt;&lt;/span&gt;&lt;span class=”c1”&gt;# reference: <a class="reference external" href="https://geoscripting-wur.github.io/Earth_Engine">https://geoscripting-wur.github.io/Earth_Engine</a>/&lt;/span&gt;
&lt;span class=”c1”&gt;# Install earthengine-api&lt;/span&gt;
pip install earthengine-api</p>
<p>&lt;span class=”c1”&gt;# Authenticate earth engine&lt;/span&gt;
earthengine authenticate
&lt;span class=”c1”&gt;# Follow procedure to authenticate and paste the access token in your terminal&lt;/span&gt;</p>
<p>&lt;span class=”c1”&gt;# Check if earth engine has been installed&lt;/span&gt;
python -c &lt;span class=”s2”&gt;&amp;quot;import ee; ee.Initialize()&amp;quot;&lt;/span&gt;
&lt;span class=”c1”&gt;# If you don't get an error, you are good to go&lt;/span&gt;
&lt;/pre&gt;&lt;/div&gt;
